# Mintlify Landing Page – UI Recreation

##  Project Overview

This project is a **UI recreation of the Mintlify landing page**, focusing on layout structure, visual hierarchy, typography, and color consistency.\
The goal was to closely match the original design while using clean, semantic HTML and modern CSS practices.

---

## Sections Recreated

The following sections from the original Mintlify website were recreated:

### 1. **Header / Navigation Bar**

- Logo placement on the left

- Navigation links (Resources, Documentation, Customers, Blog, Pricing)

- Call-to-action buttons (Contact sales, Start for free)

- Sticky-style layout behavior

### 2. **Hero Section**

- Main headline: *“The Intelligent Knowledge Platform”*

- Supporting description text

- Primary CTA button (“Start now”)

- Illustrated cloud-style background

- Center-aligned content with strong visual focus

### 3. **Quickstart / Documentation Preview**

- Sidebar-style documentation navigation

- Quickstart cards with icons

- Two-column content layout

- Clean card UI with subtle borders and shadows

### 4. **Trusted By / Partners Section**

- Company logos (Anthropic, Coinbase, Microsoft, PayPal, etc.)

- Horizontal logo alignment

- Muted grayscale styling for logos

### 5. **Features Section**

- “Built for the intelligence age”

- Feature cards explaining:

  - Build for people and AI

  - Self-updating knowledge management

- Icon-based feature highlights

- Grid-based layout

### 6. **Assistant Preview Section**

- Dashboard-style UI mockups

- Analytics cards and assistant widgets

- Light background with card separation

### 7. **Enterprise Section**

- Enterprise-focused messaging

- Compliance and access control highlights

- Dark themed feature card with stats

### 8. **Industry Use Cases**

- Card-based layout for different industries

- Image thumbnails with titles and short descriptions

- Carousel-style navigation indicators

### 9. **Call to Action Section**

- “Make documentation your winning advantage”

- Primary and secondary CTA buttons

### 10. **Footer**

- Multi-column footer links

- Legal and company links

- Brand logo and system status indicator

---

## Fonts Used

The typography closely matches Mintlify’s clean and modern style:

- **Primary Font:** `Inter`

  - Used for headings, body text, buttons, and navigation

  - Chosen for its readability and modern SaaS feel

**Fallbacks:**

```
font-family: "Inter", system-ui, -apple-system, BlinkMacSystemFont, sans-serif;
```

##  Design & Development Notes

- Semantic HTML elements were used (`header`, `main`, `section`, `footer`)

- Layout built using **Flexbox and CSS Grid**

- Spacing and alignment follow a consistent 8px system

- UI components are modular and reusable

- Focused on **visual accuracy**, not backend functionality

---

## Reference

This UI recreation is based on the official **Mintlify landing page** design, used strictly for **educational and practice purposes**.
